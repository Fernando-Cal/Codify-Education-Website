Syllabus

Overview

This 8-week Full Stack Engineering program is an accelerated introduction to modern web development. You will learn how the web works, how to build interactive frontends, how to design and implement backends, and how to connect them into a full stack system.

We use JavaScript/TypeScript, React, Node.js/Express, and PostgreSQL. Mastery of these tools is valuable, but our focus is understanding how the layers interact so you can quickly adapt to new technologies.

Prerequisites

Prior coding experience is recommended (loops, functions, conditionals). Students should also be comfortable with:

  • Using the terminal
  • Installing and configuring software
  • Following technical documentation

Weeks 1–2 are especially demanding. If you struggle, consider pausing to review fundamentals before continuing.

Alternative Preparation

Students new to coding may benefit from resources like freeCodeCamp’s JavaScript Basics or Harvard CS50x. If your interest is more data-oriented, an intro data science course may be a better fit.

Course Format

The course combines lectures, labs, discussions, and projects:

  • Lectures: Introduce concepts and walk through examples.
  • Labs: Weekly hands-on coding assignments. Attendance required.
  • Discussions: Peer collaboration, Q&A, and code review habits.
  • Projects: Three milestone builds leading to a deployed capstone project.

Weekly Schedule

Week 1 – The Web & Frontend Basics

HTTP, DNS, client/server, request/response. HTML5 semantics, CSS basics, responsive design with flexbox/grid.

Lab: Personal website with semantic HTML and responsive CSS.

Week 2 – JavaScript Foundations

ES6+ syntax, variables, functions, scope. DOM manipulation, events, forms. Debugging with DevTools.

Lab: Build an interactive to-do list app.

Week 3 – Modern Frontend (React I)

React fundamentals: components, JSX, props, state. Component hierarchies and event handling.

Lab: Simple React app. Project 1 starts: React single-page app.

Week 4 – React II

Hooks, Context API, routing, API fetching, state management patterns.

Lab: Blog frontend with routing and data fetching. Project 1 due.

Week 5 – Backend Foundations

Node.js runtime, Express basics, routing, middleware, REST APIs and JSON.

Lab: API server for blog site. Project 2 starts: API backend.

Week 6 – Databases

Relational vs non-relational databases, PostgreSQL basics, queries and joins.

Lab: Store and retrieve blog posts with PostgreSQL. Project 2 due.

Week 7 – Full Stack Integration

Authentication and authorization (sessions, JWT). Connecting React frontend with Express backend. Error handling and testing.

Lab: Users can sign up, log in, and create posts. Project 3 starts.

Week 8 – Deployment & Capstone

Cloud deployment (Heroku, Vercel, or similar), environment variables, configuration.

Capstone Project: Deploy a full stack app with authentication, CRUD, and database persistence.

Assignments

Weekly labs, weekly homework, and three projects:

  • Project 1: React Frontend (Weeks 3–4)
  • Project 2: API Server + Database (Weeks 5–6)
  • Project 3: Full Stack Capstone (Weeks 7–8)

Grading

Total: 100 points

  • Labs: 20
  • Homework: 15
  • Projects: 45 (15 each)
  • Participation: 10
  • Final Demo/Presentation: 10

Late Policy

Labs: no credit if late. Homework: 50% credit if within 48 hours. Projects: 75% credit if within 48 hours, otherwise no credit.

Citizenship & Collaboration

Collaboration is encouraged; copying is prohibited. Discuss concepts and designs freely, but write your own code (except with your project partner). AI tools (ChatGPT, Copilot, etc.) are not allowed for labs, homework, or projects.

A Parting Note

This course moves quickly. Treat each week seriously, practice consistently, and don’t fall behind. The goal is to think like an engineer: breaking problems down, testing ideas, and building scalable systems.
